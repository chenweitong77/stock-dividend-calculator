## 📱 鸿蒙 App 开发指南

### 1. 导入 DevEco Studio

1. 打开 DevEco Studio
2. `File → Open` 选择 `entry/` 目录
3. 等待 Gradle 同步完成

### 2. 数据接口（已验证可用）

| 数据 | 来源 | 接口 |
|------|------|------|
| 实时股价 | 新浪财经 | `https://hq.sinajs.cn/list=sh600519` |
| 分红数据 | 东方财富 | `https://datacenter-web.eastmoney.com/api/data/v1/get?reportName=RPT_LICO_FN_CPD` |

### 3. 核心计算公式

```
股息率 = (中报每股派息 + 年报每股派息) / 成本价 × 100%
```

**重要：分红按 PAYYEAR（利润归属年度）归类，而非按派息日期**

### 4. 运行调试

1. 连接华为手机或打开模拟器
2. 点击 `Run → Run 'entry'`
3. 输入股票代码、成本价、持仓数量
4. 点击「开始计算」
